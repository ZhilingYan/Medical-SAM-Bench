Abstract

SAMed-2 is a new foundation model for medical image segmentation built upon the SAM-2 architecture. Specifically, we introduce a temporal adapter into the image encoder to capture image correlations and a confidence-driven memory mechanism to store high-certainty features for later retrieval. This memory-based strategy counters the pervasive noise in large-scale medical datasets and mitigates catastrophic forgetting when encountering new tasks or modalities.

Method

SAMed-2 builds upon SAM-2's architecture with key innovations:

  • Temporal Adapter: Captures image correlations in the image encoder
  • Confidence-Driven Memory: Stores high-certainty features for later retrieval
  • MedBank: A large scale medical datasets

MedBank Dataset

MedBank is a comprehensive collection of various public medical imaging datasets for training and evaluating medical image segmentation models. It covers diverse modalities and anatomical structures.
